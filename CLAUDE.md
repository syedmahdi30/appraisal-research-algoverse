# CLAUDE.md — Cross-Modal Appraisal Transfer

## WHY — Project overview
- This repo implements cross-modal appraisal transfer experiments:
  - Replicate text-side appraisal probes and steering (Tak et al. 2025 style) on Gemma-3-4B via TransformerBridge.
  - Transfer learned appraisal directions to multimodal VLM behaviour on images.
- Goal: reproducible, paper-quality experiments with clear separation between:
  - Text-only replication
  - Cross-modal appraisal transfer on images
  - Verification runs on alternate VLMs.

## WHAT — Tech stack and layout
- Language: Python (>=3.11), PyTorch, Hugging Face Transformers, TransformerBridge 3.x.
- Core models:
  - Primary: `google/gemma-3-4b-it` via `TransformerBridge.boot_transformers(...)`.
  - Optional verification: Qwen3-VL or Qwen2.5-VL via raw HF hooks.
- Key directories:
  - `src/bridge/` — bridge boot + hook utilities.
  - `src/experiments/` — scripts for Stage A/B/C experiments.
  - `src/probes/` — ridge probe training and evaluation.
  - `src/data/` — dataset loaders and preprocessing.
  - `docs/` — experiment design, datasets, and model notes.
  - `.claude/` — rules, skills, and hooks configuration.
